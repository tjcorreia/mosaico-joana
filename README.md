# Mosaico - 50 Anos Joana

Visualizador com zoom do mosaico de fotos criado para os 50 anos da Joana, composto por centenas de fotografias mais pequenas.

## Ver online

https://tjcorreia.github.io/mosaico-joana/

## Como usar

- Roda do rato (ou pinça em telemóvel) para fazer zoom
- Arrastar para navegar pela imagem

## Tecnologia

Construído com [OpenSeadragon](https://openseadragon.github.io/), uma biblioteca JavaScript para visualização de imagens em alta resolução com zoom suave.
